# 3DGaussainSplatAssetGeneratorForPython
This repository is created for python users to generate the same 3D Gaussain Splat Asset that the third-party UnityGaussainSplatting package created,which can only be created from the Unity Editor menu.Which means that you can now create your own GaussainSplatAsset in RunTime application scenes or other outside Server by deploying this repo in your device.
